Inicio

Bienvenidos al curso de Diseños de Sitios Web de la UNAD

En esta OVA vamos a indagar en la temática de GIT y GITHUB

GIT es un software de control de versiones de código abierto, multiplataforma en donde se pueden crear repositorios locales en varios los sistemas operativos.

GITHUB es un servicio de hospedaje para proyectos gestionados por el sistema de control de versiones Git. GITHUB ayuda a compartir el código fuente de una manera más fácil.
